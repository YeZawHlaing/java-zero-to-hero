# purpose of sliding window
### Usage

- Array/List Processing: To solve problems related to subarrays or subsequences, such as finding the maximum sum of a subarray of a certain size.

- String Manipulation: To solve problems related to substrings, such as finding the longest substring without repeating characters.

---

### Sliding window algorithm

1. Determine Window Size: Decide on a fixed window size that defines the number of elements to consider at each step.

2. Initialize and Process: Perform initial calculation with initial elements in the window.

3. Slide the Window: Iterate through the data, slide the window one element at a time by adding the next element and removing the leftmost one.

4. Update and Evaluate: Do the calculation for current window and check if it meets the condition or not.

5. Continue Sliding: Repeat the sliding, updating, and evaluation steps until the end of the data is reached.

6. Return Result: Return the result or outcome based on the processed windows.
